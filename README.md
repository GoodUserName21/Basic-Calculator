# 🧮 Basic Calculator

A simple, interactive calculator built with HTML, CSS, and JavaScript.

## ✨ Features

- Clean and modern user interface with gradient background
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Decimal number support
- Delete and Clear buttons for easy correction
- Division by zero protection
- Smooth button interactions and animations
- Fully functional and responsive

## 🎮 How to Use

1. **Open the calculator**: Open `index.html` in your web browser
2. **Enter numbers**: Click number buttons (0-9) to enter values
3. **Choose operation**: Click operator buttons (+, -, *, /)
4. **Calculate**: Click the equals (=) button to see the result
5. **Clear**: Use the Clear (C) button to reset everything
6. **Delete**: Use the Delete (DEL) button to remove the last digit
7. **Decimals**: Click the (.) button for decimal numbers

## 📁 Files

- `index.html` - HTML structure of the calculator
- `style.css` - Styling, layout, and animations
- `script.js` - Calculator logic and functionality

## 🚀 Installation

Simply clone this repository and open `index.html` in your browser. No dependencies required!

```bash
git clone https://github.com/GoodUserName21/Basic-Calculator.git
cd Basic-Calculator
open index.html
```
